Coordenadas Relativas

Hay varias formas de ingresar coordenadas relativas. Esto puede ser una forma más sencilla de especificar puntos en lugar de introducir coordenadas absolutas. Las coordenadas relativas siempre requieren que un punto anterior esté disponible como punto de referencia. Esto generalmente se logra ingresando el primer punto en un comando.

Coordenadas Ortográficas

Las coordenadas relativas siempre se especifican mediante el símbolo '@'.

Coordenadas Polares

Para coordenadas polares, los ángulos se denotan con el símbolo '<'. La coordenada del ángulo se puede ingresar antes o después de la coordenada del radio. Una simplificación adicional es que se puede usar el símbolo del ángulo en lugar de la coma entre las coordenadas.

Ejemplo de Coordenadas Polares: @4,<30
Ejemplo de Angulo Simplificado: @2<-15
Ejemplo de Angulo Antes: @<270,4
