Change the scale of design geometry.

Steps

  1. Select objects to scale
  2. Type ${action.command}
  3. Select an anchor point
  4. Select a reference point
  5. Move the reference point to choose a new scale
  6. Select the target point
  7. The objects are modified to the new scale

Details

This command changes the scale of selected design geometry. The command requires that geometry be selected before starting the command. Please refer to selecting geometry for more information.

Once you have selected the design geometry you want to scale, start the command by typing ${action.command}.

You are then prompted to enter an anchor point. The anchor point will stay stationary while the geometry is scaled around it. For example, if you want to scale a rectangle and keep the lower left anchored where it is, select the lower left point on the rectangle as the anchor.

You are then prompted to enter a reference point. The reference point is used to determine both the new scale value and direction of the scale will be applied. Moving the reference point toward the anchor will make the geometry smaller and moving the reference point away from the anchor will make the geometry larger. It is even possible to mirror the geometry by moving the reference point past the anchor point.

Once the reference point is selected then the design geometry is resized according to the new scale.
