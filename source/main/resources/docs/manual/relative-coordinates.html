Relative Coordinates

There are several ways to enter relative coordinates. This can be a simpler way to specify points rather than entering absolute coordinates. Relative coordinates always require that a prior point be available as the reference point. This is usually accomplished by entering the first point in a command.

Orthographic Coordinates

Relative coordinates are always specified by using the '@' symbol.

Polar Coordinates

For polar coordinates, angles are denoted by the '<' symbol. The angle coordinate may be entered before or after the radius coordinate. A further simplification is that the angle symbol may be used in place of the comma between the coordinates.

Polar Coordinate Example: @4,<30
Simplified Angle Example: @2<-15
Angle First Example: @<270,4
